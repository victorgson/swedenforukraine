## Tech stack
- Swift 5.8
- SwiftUI

- Xcode 14.3
- Firestore

Architecture 
- MVVM
- Repository pattern

# Installation
- Clone the project
- Wait for indexing of files & packages.
- Run the project, there is three different targets, one for each language. (Swedish - Default, Ukrainian and English)
